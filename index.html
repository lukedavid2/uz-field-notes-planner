Field Notes — Content Planner

Schedule and preview the Undercover Zest blog. Posts you date here publish themselves to undercoverzest.app on the day you choose.

Connect to GitHub

  1. Open github.com → fine-grained tokens → Generate new.
  2. Resource owner lukedavid2; Repository access → Only select repositories → undercover-zest.
  3. Permissions → Repository permissions → Contents → Read and write. Nothing else.
  4. Generate, copy the token, and paste it below. It is stored only in this browser.

The planner reads and writes one file — queue.json — on a private content-pipeline branch. It never touches your live site directly; the publish step is a separate scheduled GitHub Action.
