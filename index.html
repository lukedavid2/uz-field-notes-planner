Field Notes — Content Planner

Write, schedule and preview the Undercover Zest blog. Posts you date here publish themselves to undercoverzest.app on the day you choose.

Connect to GitHub

  1. Open github.com → fine-grained tokens → Generate new.
  2. Resource owner lukedavid2; Repository access → Only select repositories → undercover-zest.
  3. Permissions → Repository permissions → Contents → Read and write. Nothing else.
  4. Generate, copy the token, and paste it below. It is stored only in this browser.

The planner reads and writes the content-pipeline branch — the queue.json schedule and the article files. It never touches your live site directly; publishing is a separate scheduled GitHub Action.
